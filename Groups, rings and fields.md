## Definition group

A group is a tuple (G, #), where G is a set and # is a binary operation, that fufils the following properties:
- (a#b)#c = a#(b#c) ... associativity
- $\exists$ e : a#e = a ... neutral element
- $\forall$ a $\in$ G $\exists$ b : a#b = e ... inverse element
